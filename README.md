# Uber Eats 週間売上シミュレーター

Uber Eats配達パートナー向けの週間売上シミュレーターです。
スマホのホーム画面に追加して、アプリのように使用できます。

## 機能

- 📊 基本設定（平均単価、1時間あたりの配達件数）
- 🏆 クエスト設定（月～木、金～日で別々に設定可能）
- 📅 各曜日の稼働時間・件数・金額入力
- 💰 週間売上の自動計算
- 🎯 目標金額設定と達成までの時間表示
- 💾 入力データの自動保存（localStorage使用）
- 📆 **過去の売上履歴カレンダー**（月単位表示、月曜日始まり）
- 📝 **日付単位のデータ入力**（カレンダーの日付をタップして個別入力）
- 💰 **日ごとの詳細管理**（稼働時間、配達件数、基本売上、クエスト報酬）
- 📊 **月間集計表示**（自動計算で月の合計売上を表示）
- 🔄 **週の自動切り替わり**（日曜日→月曜日で自動保存＆クリア）
- 💾 **手動保存機能**（任意のタイミングで週データを保存可能）
- 💸 **経費管理機能**（ガソリン代、車両整備費、駐車場代など）
- 📸 **レシート写真保存**（写真をアプリ内に保存、確定申告用）
- 📊 **CSV出力機能**（経費データをCSVでエクスポート、確定申告対応）

## ホーム画面への追加方法

### iPhone (Safari)
1. Safariでindex.htmlを開く
2. 画面下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

### Android (Chrome)
1. Chromeでindex.htmlを開く
2. 画面右上のメニュー（⋮）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

## 使い方

1. **設定ボタン**から平均単価と配達件数を設定
2. **クエストボタン**から月～木、金～日のクエストを設定
3. **目標金額**を入力（任意）
4. 各曜日の**稼働時間**を入力
   - 時間のみ入力：予想金額を自動計算
   - 時間・件数・金額すべて入力：実績値を反映
5. 週間売上予測が自動で表示されます
6. **カレンダーボタン**から過去の売上データをカレンダーで確認できます
   - 月単位のカレンダービューで過去のデータを表示
   - データがある日は紫色でハイライト表示
   - 日付をタップして詳細を入力・編集できます
7. **日付タップで個別データ入力**
   - カレンダーの任意の日付をタップ
   - 稼働時間、配達件数、基本売上、クエスト報酬を個別入力
   - 過去のデータも後から追加・編集可能
8. **月間集計**
   - カレンダー下部に月の合計が自動表示
   - 稼働時間、配達件数、基本売上、クエスト、合計売上を一覧
9. **経費ボタン**から経費管理機能を利用できます
   - 新規経費を追加：日付、カテゴリー、品目、金額、写真を入力
   - カテゴリー：ガソリン代、車両整備、駐車場代、通信費、保険料、装備品、その他
   - **借方科目（勘定科目）**：車両費、燃料費、修繕費、地代家賃など12科目から選択
   - **事業使用割合**：プライベート共用の場合は事業使用分の割合を設定（0-100%）
   - レシート写真：カメラやギャラリーから画像を選択して保存
   - 月間集計：当月の経費合計が自動表示（事業使用割合考慮済み）
   - CSVエクスポート：確定申告用にデータをCSV形式でダウンロード（借方科目・事業割合含む）
10. 週のデータは以下のタイミングで自動保存されます：
    - 日曜日にアプリを閉じて、月曜日に再度開いたとき
    - または「今週のデータを保存してクリア」ボタンで手動保存

## 技術仕様

- **フレームワーク**: React 18 (CDN)
- **スタイリング**: Tailwind CSS
- **PWA対応**: Service Worker、manifest.json
- **データ保存**: localStorage
- **オフライン対応**: あり

## ファイル構成

- `index.html` - メインアプリケーション
- `manifest.json` - PWA設定
- `sw.js` - Service Worker

## ブラウザ対応

- Chrome / Edge (最新版)
- Safari (iOS 11.3+)
- Firefox (最新版)

## 注意事項

※実際の売上は配達エリアや時間帯により変動します。
※本シミュレーターの計算結果は目安であり、実際の収入を保証するものではありません。
