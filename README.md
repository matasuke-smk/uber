# Uber Eats 週間売上シミュレーター

Uber Eats配達パートナー向けの週間売上シミュレーターです。
スマホのホーム画面に追加して、アプリのように使用できます。

## 機能

- 📊 基本設定（平均単価、1時間あたりの配達件数）
- 🏆 クエスト設定（月～木、金～日で別々に設定可能）
- 📅 各曜日の稼働時間・件数・金額入力
- 💰 週間売上の自動計算
- 🎯 目標金額設定と達成までの時間表示
- 💾 入力データの自動保存（localStorage使用）
- 📆 **過去の売上履歴表示**（週単位、月曜日始まり）
- 🔄 **週の自動切り替わり**（日曜日→月曜日で自動保存＆クリア）
- 💾 **手動保存機能**（任意のタイミングで週データを保存可能）

## ホーム画面への追加方法

### iPhone (Safari)
1. Safariでindex.htmlを開く
2. 画面下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

### Android (Chrome)
1. Chromeでindex.htmlを開く
2. 画面右上のメニュー（⋮）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

## 使い方

1. **基本設定ボタン**から平均単価と配達件数を設定
2. **クエスト設定ボタン**から月～木、金～日のクエストを設定
3. **目標金額**を入力（任意）
4. 各曜日の**稼働時間**を入力
   - 時間のみ入力：予想金額を自動計算
   - 時間・件数・金額すべて入力：実績値を反映
5. 週間売上予測が自動で表示されます
6. **履歴ボタン**から過去の週の売上データを確認できます
   - 週単位で売上、稼働時間、配達件数を表示
   - 詳細表示で各曜日のデータも確認可能
7. 週のデータは以下のタイミングで自動保存されます：
   - 日曜日にアプリを閉じて、月曜日に再度開いたとき
   - または「今週のデータを保存してクリア」ボタンで手動保存

## 技術仕様

- **フレームワーク**: React 18 (CDN)
- **スタイリング**: Tailwind CSS
- **PWA対応**: Service Worker、manifest.json
- **データ保存**: localStorage
- **オフライン対応**: あり

## ファイル構成

- `index.html` - メインアプリケーション
- `manifest.json` - PWA設定
- `sw.js` - Service Worker

## ブラウザ対応

- Chrome / Edge (最新版)
- Safari (iOS 11.3+)
- Firefox (最新版)

## 注意事項

※実際の売上は配達エリアや時間帯により変動します。
※本シミュレーターの計算結果は目安であり、実際の収入を保証するものではありません。
